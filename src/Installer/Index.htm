Overview

Lift Bridge is a plugin for FieldWorks Language Explorer which permits sharing of LIFT dictionary data with remote colleagues such as WeSay.

Usage

See documentation for details on how to use Lift Bridge.

Downloads

Lift Bridge is available for use with two versions of FieldWorks: 6.0.6 and 7.0. Select the version of Lift Bridge to download based on which version of FieldWorks you have installed.
